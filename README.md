# Casa Dolce

SPA de catálogo para una pastelería artesanal premium. El sitio muestra productos, galería, feed visual de Instagram, testimonios y un formulario simple para preparar cotizaciones por WhatsApp.

## Ejecutar el proyecto

Instala dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm run dev
```

Genera build de producción:

```bash
npm run build
```
